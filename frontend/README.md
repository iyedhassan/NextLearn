# NextLearn Frontend

A premium React frontend for the NextLearn E-learning platform.

## Technologies
- React 19
- Vite
- Axios
- React Router Dom
- React Icons

## Getting Started

1. **Install dependencies**:
   ```bash
   npm install
   ```

2. **Run the development server**:
   ```bash
   npm run dev
   ```

3. **Backend Setup**:
   Ensure your Laravel backend is running at `http://localhost:8000`. If you use a different URL, update `src/api/axios.js`.

## Features
- Premium Glassmorphism UI
- Responsive Navbar
- Integrated Authentication (Login/Register)
- Axios pre-configured for Laravel Sanctum
